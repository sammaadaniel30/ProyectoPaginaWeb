Intro. Guía

Mecánicas del juego

Este vídeo es el tráiler del juego en el cual se muestra lo que se tiene que hacer.

El vídeo en sí está en inglés, cabe resaltar que no hay versión en castellano aún de este juego.

Aun así este vídeo, visualmente, da la idea general de lo que trata el juego.

Controles

Movimientos básicos

Permite desplazarse por el entorno utilizando las teclas W, A, S y D para moverse en todas las direcciones.

Sprint

Aumenta la velocidad de movimiento manteniendo pulsada la tecla Shift para escapar rápidamente de situaciones peligrosas.

Saltar y agacharse

Permite saltar obstáculos con la tecla Espacio y agacharse para ocultarse o pasar por zonas estrechas.

Interacción con objetos

La tecla E permite interactuar con objetos del entorno, recoger elementos y activar mecanismos del juego.

Inventario

Permite acceder rápidamente a los objetos del inventario mediante las teclas numéricas para seleccionar herramientas.

Modo Espectador

Permite cambiar entre jugadores al estar en modo espectador usando el ratón para observar la partida.

Expresiones del personaje (I)

Asigna teclas a diferentes expresiones faciales del personaje para comunicarse de forma visual con otros jugadores.

Expresiones del personaje (II)

Incluye expresiones adicionales como felicidad, locura o sospecha para reforzar la interacción social.

Comunicación por voz

Permite activar el chat de voz, silenciar el micrófono o hablar con otros jugadores mediante teclas asignadas.

Tutorial

Aprende a jugar paso a paso

Si todavía tienes dudas sobre cómo jugar, este vídeo tutorial te ayudará a comprender todas las mecánicas básicas del juego de forma clara y práctica.

En él se muestran ejemplos reales de una partida, explicando movimientos, interacción con objetos y estrategias para sobrevivir desde el inicio.

Es recomendable verlo si es tu primera vez jugando o si quieres mejorar tu forma de jugar.
